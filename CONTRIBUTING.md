# Contributing to EDL

Thank you for your interest in contributing to EDL! Since the EDL project is still in an early stage of development
the contribution guide is likely to change in the future as the project matures.
Currently, the best way to get started is to reach out on the [Discord Server](https://discord.gg/syv7pCQ4ky).

# Bug Reports

You found a bug? Please create a bug report under the `issues` section including a description of the bug and a minimal reproducible example.
